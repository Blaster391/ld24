Wibblymat's Ludum Dare #24 Entry page

This page is a place to keep all of my information about Ludum Dare 24 together.

All of the interesting stuff can be found at the links on the left.

The game will be HTML5 canvas/javascript. I will use Chrome for development, so it is possible that it will only work in Chrome. Hopefully not!

I'm using crafty.js, the GIMP, Audacity, bfxr, iNudge, Sublime Text 2, git, Dropbox, imagemagick and pen and paper.

Once work begins, you will be able to play the game as I go along
